# Calculator Web Application

## Description
A calculator built using HTML, CSS, and JavaScript that performs basic arithmetic operations.

## Features
- Addition, Subtraction, Multiplication, Division
- Clear Screen (C)
- Delete Last Character (X)
- Real-Time Result Display
- Keyboard Support
- Error Handling

## Technologies Used
- HTML
- CSS
- JavaScript
